# < Introduction to SQL >
## What is SQL?
* SQL은 Structured Query Language의 약자. 구조화 질의 언어.
* 기술자 및 비기술자 사용자에게 모두 관계형 DB에서 데이터 조회, 조작 및 변환을 가능하게 하도록 설계된 언어.
* 이러한 간편성 덕분에, SQL database는 수백만의 웹 사이트와 모바일 애플리케이션을 위해 안전하고 확장 가능한 저장 공간을 제공.
* SQLite, MySQL, Postgres, Oracle and Microsoft SQL Server 등의 많은 SQL databases가 존재함.
  * 그것들 모두 이 사이트(SQLBolt)에서 가르칠 보편적인 SQL 언어 표준을 지원함.
  * 그러나 각각의 구현 방식은 지원하는 추가 기능 및 저장 유형에서 차이가 존재할 수 있음.
***
## 관계형 데이터베이스(Relational Databases)
* 관계형 데이터베이스는 서로 관련된 2차원 테이블들의 모음.
* 각각의 테이블들은 엑셀 spreadsheet와 유사하게 고정된 수의 명명된 열(= 테이블의 속성)과 여러 개의 행으로 구성됨.
* SQL을 배우는 목적은 데이터에 대한 구체적인 질문에 답하는 방법을 익혀, 향후 더 나은 의사결정을 내리는 데 도움을 주는 것.
