# < 라이브러리 >

## 01. 라이브러리의 개념

* 라이브러리란 프로그램을 효율적으로 개발할 수 있도록 자주 사용하는 함수나 데이터들을 미리 만들어 놓은 집합체.
  * 자주 사용하는 함수들의 반복적인 코드 작성을 피하기 위해 미리 만든 것, 필요할 때 언제든지 호출해 사용함.
  * 표준 라이브러리와 외부 라이브러리가 존재.
* 표준 라이브러리: 프로그래밍 언어에 기본으로 포함된 라이브러리, 여러 종류의 모듈이나 패키지 형태.
* 외부 라이브러리: 개발자들이 필요한 기능들을 만들어 인터넷 등에 공유한 것, 외부 라이브러리 설치 후 사용.

***

## 02. Java의 주요 표준 라이브러리

* Java는 라이브러리를 패키지에 포함해 제공함.
* 각 패키지에는 Java 응용 프로그램 개발에 필요한 method들이 class로 정리되어 있음.
  * Java에서 패키지를 사용하려면 `import java.util`과 같은 형식으로 **import문**을 선언해 사용해야 함.
  * import로 선언한 패키지 내부 class의 method 사용 시 `Math.abs()`같이 **'클래스명.메소드명()'** 형식을 사용해야 함.

### 02-1. Java에서 사용할 수 있는 여러 패키지 종류

* `java.lang`
  * Java에 기본적으로 필요한 인터페이스, 자료형, 예외 처리 등에 관련된 기능을 제공.
  * import문 없이도 사용 가능.
  * 주요 클래스: String, System, Process, Runtime, Math, Error 등
* `java.util`
  * 날짜 처리, 난수 발생, 복잡한 문자열 처리 등에 관련된 기능을 제공.
  * 주요 클래스: Date, Calendar, Random, StringTokenizer 등
* `java.io`
  * 파일 입출력과 관련된 기능 및 프로토콜을 제공.
  * 주요 클래스: InputStream, OutputStream, Reader, Writer 등
* `java.net`
  * 네트워크와 관련된 기능을 제공.
  * 주요 클래스: Socker, URL, InetAddress 등
* `java.awt`
  * 사용자 인터페이스와 관련된 기능을 제공.
  * 주요 클래스: Frame, Panel, Dialog, Button, Checkbox 등

## 03. Java의 주요 메소드

### String 클래스

* `A.compareTo(B)`
  * 숫자로된 문자열 A와 B를 비교해 같으면 0, A가 크면 1, B가 크면 -1을 반환함.
* `A.equals(B)`
  * 대소문자를 구분해 문자열 A와 B를 비교한 후 같으면 참, 다르면 거짓을 반환함.
* `A.equalsIgnoreCase(B)`
  * 대소문자 구분없이 문자열 A와 B를 비교한 후 같으면 참, 다르면 거짓을 반환함.
* `toLowerCase(문자열)`
  * 문자열을 모두 소문자로 변환함.
* `toUpperCase(문자열)`
  * 문자열을 모두 대문자로 변환함.
* `split(구분자)`
  * 지정된 구분자로 문자열을 구분하여 분리함.
* `replaceAll(변환 대상, 변환할 문자)`
  * 변환 대상을 변환할 문자로 치환함.
* `A.substring(위치)`
  * 문자열 A에서 지정한 위치를 포함한 이후의 모든 문잗열을 반환함.
* `A.charAt(위치)`
  * 문자열 A에서 위치에 해당하는 문자를 반환함.
* `A.length()`
  * 문자열 A의 길이를 반환함.
* `A.trim()`
  * 문자열 A의 좌우 공백을 제거함.
* `getNumericValue()`
  * 숫자 형태의 문자열을 정수형으로 반환함.
    * '0 ~ 9': 숫자 형태로 반환함.
    * 'a/A ~ z/Z': 10 ~ 35까지의 값을 반환함.
    * 특수 문자: -1을 반환함.
    * 분수처럼 정수형으로 표현할 수 없는 문자열: -2를 반환함.

### Math 클래스

* `Math.abs()`
  * 절댓값을 반환함.
* `Math.sqrt()`
  * 제곱근 값을 반환함.
* `Math.log10()`
  * 인수가 10을 몇 번 곱해야 나오는 수인지를 구함.
* `Math.pow()`
  * 거듭제곱한 값을 반환함.
* `Math.max()`
  * 최댓값을 반환함.
* `Math.min()`
  * 최솟값을 반환함.
* `Math.round()`
  * 소수점 첫째 자리에서 반올림한 값을 반환함.
* `Math.ceil()`
  * 소수점을 올림한 값을 반환함.

### StringTokenizer 클래스

* `countTokens()`
  * StringTokenizer 객체의 토큰 개수를 반환함.
* `hasMoreTokens()`
  * StringTokenizer 객체에 반환할 토큰이 있으면 참, 없으면 거짓을 반환함.
* `nextToken()`
  * StringTokenizer 객체에서 차례로 토큰을 가져와 반환함.

## 04. Python의 주요 표준 라이브러리

* Python도 Java처럼 라이브러리를 패키지로 제공하며, 그 패키지에는 응용 프로그램 개발에 필요한 클래스 및 메소드들이 정의되어 있음.
  * 라이브러리 사용 시 `import random`과 같이 선언하고, 메소드 사용 시 `random.choice()`와 같이 사용함.

### 04-1. Python의 주요 클래스

* 내장함수
  * Python의 기본적인 인터페이스
  * import문이나 클래스명 없이도 사용 가능.
  * 주요 메소드: `abs()`, `slice()`, `pow()`, `print()` 등
* os
  * 운영체제와 상호작용 하기 위한 기능 제공.
  * 주요 메소드: `getcwd()`, `chdir()`, `system()` 등
* re
  * 고급 문자열 처리를 위한 기능 제공.
  * 주요 메소드: `findall()`, `sub()` 등
* math
  * 복잡한 수학 연산을 위한 기능 제공.
  * 주요 메소드: `cos()`, `log()` 등
* random
  * 무작위 선택을 위한 기능 제공.
  * 주요 메소드: `choice()`, `sample()`, `random()`, `randrange()` 등
* statistics
  * 통계값 산출을 위한 기능 제공.
  * 주요 메소드: `mean()`, `median()`, `variance()` 등
* datetime
  * 날짜와 시간 조작을 위한 기능 제공.
  * 주요 메소드: `today()`, `date()`, `strftime()` 등

## 05. Python의 주요 메소드

### 문자열 관련

* `upper()`: 문자열을 대문자로 변경함.
* `lower()`: 문자열을 소문자로 변경함.
* `capitalize()`: 문자열의 첫번째 글자는 대문자, 나머지는 모두 소문자로 변경함.
* `title()`: 문자열에서 각 단어의 첫번째 글자만 대문자로 변경함.
* `replace(값1, 값2)`: 문자열에서 값1을 찾아서 값로 변경함.
* `split(값)`: '값'을 기준으로 문자열을 분리하여 리스트로 반환하며, '값'을 생략하면 공백으로 문자열을 분리함.
* `count(값)`: 문자열에서 '값'을 검색하여 '값'의 개수를 반환함.
* `find(값)`: 문자열에서 처음 검색되는 '값'의 위치를 반환하고, 찾지 못하면 -1을 반환함. (검색 시작 위치는 인덱스 0)
* `index(값)`: 문자열에서 처음 검색되는 '값'의 위치를 반환하고, 찾지 못하면 에러 발생.
* `len()`: 문자열의 길이를 반환함.

### 리스트 관련

* `list()`: 반복 가능한 객체를 리스트 자료형으로 변환 또는 빈 리스트 생성.
* `len()`: 리스트의 요소의 개수를 반환.
* `pop(위치)`: 리스트의 '위치'에 있는 값을 출력하고 해당 요소를 삭제, '위치' 생략 시 마지막 요소를 삭제.
* `remove(값)`: 리스트에서 '값'을 찾아 해당 요소를 삭제.
* `count(값)`: 리스트에서 '값'이 저장되어 있는 요소들의 개수를 반환.
* `extend(리스트)`: 리스트의 끝에 새 리스트를 추가해 확장함.
* `reverse()`: 리스트의 순서를 역순으로 변환.
* `copy()`: 리스트를 복사.
* `index(값)`: 리스트에서 '값'이 저장된 요소의 위치 반환.
* `sort()`: 리스트를 정렬, 기본값은 오름차순.
  * `reverse=True`이면 내림차순, `reverse=False`이면 오름차순
* `sum(리스트)`: 리스트의 모든 요소의 합을 계산해 반환.

### 딕셔너리 관련

* `get(key, value)`: key에 해당하는 값을 반환.
  * key가 없으면 None을 반환하거나 default로 지정된 값을 반환.
* `keys()`: 딕셔너리의 모든 키를 반환.
* `values()`: 딕셔너리의 모든 값을 반환.
* `items()`: 딕셔너리의 모든 (키, 값) 쌍을 반환.
* `update()`: 지정된 딕셔너리의 키-값 쌍을 현재 딕셔너리에 갱신, 지정된 키-값 쌍이 없으면 뒤쪾에 추가.
* `setdefault()`: 키-값 쌍을 추가.
* `pop()`: key에 해당하는 항목을 제거 및 그 값을 반환.
* `copy()`: 딕셔너리를 복사.
* `clear()`: 딕셔너리의 모든 항목을 제거.

### 세트 관련

* `set()`: 반복 가능한 객체를 세트 자료형으로 변환 또는 빈 세트를 생성.
* `len()`: 세트 요소의 개수를 반환.
* `pop()`: 세트의 값을 출력 및 요소 삭제.
* `add(값)`: 세트에 값을 추가하고, 추가할 값이 이미 존재하면 추가하지 않음.
* `update(세트)`: 세트에 새로운 세트를 추가해 확장.
* `remove(값)`: 세트에서 값을 찾아 해당 요소를 삭제.

## 06. Java와 Python의 공통된 수학 함수

* `pow(x, y)`
  * x의 y승을 구함.
  * `System.out.printf("%f", pow(2, 3))`
* `sqrt(x)`
  * x의 제곱근을 구함.
  * `System.out.printf("%f", sqrt(9))`
* `abs(x)`
  * x의 절댓값을 구함.
  * `System.out.printf("%d", abs(-5))`
* `log(x)`
  * 자연 로그에 대한 값을 구함.
  * `System.out.printf("%f", log(3))`
* `log10(x)`
  * 상용 로그에 대한 값을 구함.
  * `System.out.printf("%f", log10(100))`
* `ceil(x)`
  * 소수점을 올림해 정수로 표현함.
  * `System.out.printf("%d", ceil(3.1415))`

## 07. Java와 Python의 공통된 난수 함수

* `random()`
  * 0에서 1 사이의 실수 난수를 구함.
  * (Java) `a = (int)(Math.random() * 10 + 1);`
  * (Python) `a = int(random.random() * 10 + 1);`
