# < 객체지향 프로그래밍 언어 >

---

## 01. 객체지향 프로그래밍 언어

* 객체지향 프로그래밍 언어는 현실 세계의 개체를 기계의 부품처럼 하나의 객체로 만들어, 기계의 부품들을 조립해 제품을 만들 듯이 SW를 개발할 때 객체들을 조립해 프로그램을 작성하는 기법.
  * 프로시저보다는 명령과 데이터로 구성된 객체를 중심으로 하는 프로그래밍 기법.
  * 한 프로그램을 다른 프로그램에서 이용할 수 있도록 함.
  
---

## 02. 객체지향 프로그래밍 언어의 종류

### 02-1. Java

* 분산 네트워크 환경에 적용 가능.
* 멀티스레드 기능을 제공하여 여러 작업을 동시에 처리 가능.
* 운영체제 및 하드웨어에 독립적이며, 이식성이 강함.

### 02-2. C++

* C언어에 객체지향 개념을 적용한 언어.
* 모든 문제를 객체로 모델링하여 표현.

### 02-3. Smalltalk

* 1세대 객체지향 프로그래밍 언어
* 순수한 객체지향 프로그래밍 언어
* 최초로 GUI를 제공함.
